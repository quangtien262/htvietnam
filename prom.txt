cụ thể page đó đang config ở line 46 trong file routes/web.php là config route cho admin: Route::get('aio/{any?}', [AIOController::class, 'dashboard'])->name('dashboard')->where('any', '.*');

Tài khoản của khách hang thì được bắt đầu là "user/" ở line 39 trong file routes/web.php là config route cho user, là trang profile của khách hang:
em hãy xem ở file vite.config.ts đ biêt config của hệ thống nhé

+) các route được khai báo trong 'resources/common/route.tsx' thông qua biến ROUTE, sau đó truyền vào trong route ở file 'resources/js/app.tsx'
+) các api ở frontend được đăng ký file api.tsx sau đó gọi ra sử dung  ở  những nơi cần.
+) Các link ở frontend hãy đăng ký ở file menu.jsx
+) route api của Laravel được lưu ở file routes/spa_route.php , lưu ý  prefix là aio/api

/////// giai thích đa ngôn ngữ
Hãy code tính năng quản lý menu cho web đa ngôn ngữ, với các tính năng chính là quản lý danh sách (có thể sắp xếp thứ tự bang cách kéo thả dạng đa cấp, sử dung Drag treeNode của andt) thêm, sửa, xóa.
migrate của Bảng menu là: 2022_09_23_044221_create_menus_table.php
bang menu cho các ngôn ngữ tương ứng menu_data: 2023_08_20_142522_create_menus_data_table.php
- Giải thích ý nghĩa các trường trong bang menu như sau:
+) name: tiêu đề menu
+) images: hình ảnh liên quan đến menu, cho phép upload nhiều hình ảnh, anh định lưu vào database nhiều  đường dẫn của hình anh dạng json, em thấy vậy hợp lý không
+) parent_id: nó chính là id của menu cha, như vậy đối với cấp cao nhất thì menu sẽ là bang 0, và các menu con sẽ là id của menu cha tương ứng .
+) current_link: string
+) icon là chọn từ danh sách icon trong comp_icon.tsx
+) is_active:checkbox
+) is_front: nhập dạng checkbox
+) create_by: là lưu ngầm tên người tạo trên backend

- Giải thích bang menu_data như sau
name_data: tiêu đề menu theo ngôn ngữ.
name_data_description: mô tả cho tiêu đề.
data_id: là id của menu tương ứng,
languages_id: là id của ngôn ngữ tương ứng, với mỗi ngôn ngữ anh định là sẽ để 1 tab riêng. danh sách ngôn ngữ thì sẽ lấy ở bang languages (2023_10_16_150830_create_languages_table.php)
description,
content,
meta_description, meta_keyword: 2 field này nhập lieu dnagj textarea, mục đich là để tối ưu cho SEO website sau này
